# Joenil P. Acero - Personal Portfolio

A modern, responsive, and minimalist portfolio website for **Joenil P. Acero**, a 4th-year BSIT student and aspiring Full-Stack & SaaS Developer.

## 🚀 Features

- **Fully Responsive**: Optimized for mobile, tablet, and desktop devices.
- **Dark Mode Support**: Persists user preference via `localStorage` and respects system settings.
- **Modern UI/UX**: Built with a clean, minimalist aesthetic using the Inter font family.
- **Project Showcase**: Highlights key projects with detailed descriptions and tech stacks.
- **Interactive Elements**: Smooth scrolling, hover effects, and a theme toggle.
- **Single File Architecture**: Minimal dependencies, using inline CSS and JS for fast loading.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Modern Flexbox and Grid layouts.
- **JavaScript**: Vanilla JS for theme switching and interactivity.
- **FontAwesome**: High-quality icons.
- **Google Fonts**: Inter typography.
